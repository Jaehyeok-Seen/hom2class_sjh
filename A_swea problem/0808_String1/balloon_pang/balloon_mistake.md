🔄 반복 학습 체크리스트

 2차원 리스트 인덱싱: arr[i][j] 형태로 정확히 사용
 변수 선언 위치: 전역 vs 지역 스코프 고려
 배열 경계 체크: < N (이하가 아닌 미만)
 들여쓰기 일관성: 중첩 레벨에 맞는 적절한 indentation

 1. 경계값 처리 패턴
# 범위 체크 후 안전한 접근
if 0 <= ni < N and 0 <= nj < M:
    # 유효한 인덱스일 때만 실행
    process(arr[ni][nj])

2. 2차원 리스트 인덱싱 실수
# ❌ 잘못된 방법
num_list[i,j]  # 튜플 인덱싱으로 인식됨

# ✅ 올바른 방법  
num_list[i][j]  # 2차원 리스트 접근
